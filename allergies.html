    This app was made while following an interactive and video-based tutorial from the "DOM Scripting by Example" module of the Team Tree House's "Intro to Programming" course. This module is taught by Guil Hernandez and Joel Kraft. Minor modifications to instructor written code were done by the student, Nataly Mota, in order to customize this app to change the kinds of things that can be listed with it, so that it is now a list of suspected allergies. Then, student Palina Harkach contributed to modifying the CSS styles in order to make the color scheme fit the new app's medical theme better, as well as to improve the formal feel of the font selection used. These modifications are still under development.
